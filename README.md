#  ISATS Ferrari v2.7.5: Pure Core Edition

"15,000개의 부품 중 단 50개만 남겼습니다. 이것은 정제(Pure)된 지능입니다."

##  개요
ISATS Ferrari는 기존의 무겁고 복잡한 트럭 시스템에서 완전히 벗어나, 초경량고성능을 추구하는 AI 자동매매 레이싱카입니다. 

##  핵심 아키텍처
- **Pure Core**: 99.7%의 불필요한 코드 소각
- **DNA Evolution**: 진화하는 매매 파라미터
- **Hybrid Intelligence**: CNN-LSTM 패턴 인식 + 전문과 직관 결합
- **Dual Engine**: 실전(REAL) + 모의(VIRTUAL) 동시 운영
